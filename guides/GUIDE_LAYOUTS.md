# Study Guide Layout System

This document defines the layout styles for each guide category. When creating a new study guide, use the layout specified for its category to ensure visual variety while maintaining consistency within categories.

---

## Layout Overview

| Category | Layout Style | Key Features |
|----------|--------------|--------------|
| **Cardiovascular** | Classic Document | Vertical TOC, comparison tables, staged content boxes |
| **Respiratory** | Classic Document | Vertical TOC, comparison tables, breathing technique visuals |
| **Neurological** | Magazine Style | Pill navigation, timelines, side-by-side comparisons, hero sections |
| **Endocrine** | Dashboard/Panel | Hormone level panels, feedback loop diagrams, value ranges |
| **Renal** | Flowchart Style | Process flows, filtration diagrams, lab value cards |
| **Gastrointestinal** | Anatomy Journey | Top-to-bottom organ flow, regional sections, pathway visuals |
| **Musculoskeletal** | Visual Atlas | Body region cards, procedure timelines, mobility checklists |
| **Pharmacology** | Reference Card | Drug class cards, comparison grids, quick-reference boxes |
| **Fundamentals/Clinical Skills** | Step-by-Step Procedural | Numbered procedures, checklist format, technique boxes |
| **Maternal-Newborn** | Timeline/Stages | Trimester/stage sections, milestone markers, assessment bands |
| **Pediatrics** | Timeline/Stages | Age-based sections, developmental milestones, growth charts |
| **Mental Health** | Therapeutic Focus | Conversation examples, intervention cards, safety protocols |
| **Lab Values** | Quick Reference | Value cards, critical ranges, color-coded normals |
| **Safety/Infection Control** | Protocol Blocks | Precaution boxes, PPE guides, isolation cards |

## Detailed Layout Specifications

### 1. CLASSIC DOCUMENT LAYOUT
**Used by:** Cardiovascular, Respiratory

**Characteristics:**
- Traditional vertical Table of Contents
- Section numbers with bordered headers
- Comparison tables (e.g., Left vs Right HF, Bronchitis vs Emphysema)
- Info boxes: Clinical Pearl, NCLEX Alert, Warning, Nursing Priority
- Staged content (Stage I, II, III, IV)
- Cover with gradient matching organ system colors

**Color Schemes:**
- Cardiovascular: Blue → Purple → Red gradient (heart/blood)
- Respiratory: Cyan → Teal → Green gradient (air/lungs)

**Key Components:**
```
- cover-page with cover-logo, cover-category, cover-title
- table-of-contents (vertical sidebar style)
- guide-section with section-header (numbered)
- comparison-table (side-by-side)
- info-box (clinical, nclex, warning, tip)
- key-points-summary
```

**Existing Examples:** heart-failure.html, copd.html, myocardial-infarction.html

---

### 2. MAGAZINE STYLE LAYOUT
**Used by:** Neurological

**Characteristics:**
- Horizontal pill navigation (sticky)
- Hero sections for key assessments (e.g., FAST)
- Visual timelines (e.g., tPA window)
- Side-by-side comparison cards (not tables)
- Inclusion/Exclusion criteria boxes
- Time-critical badges with animations
- Intervention card grids

**Color Schemes:**
- Stroke: Blue → Red gradient (ischemic vs hemorrhagic)
- Seizures: Purple → Blue gradient (brain activity)
- TBI: Dark blue → Orange gradient (trauma alert)

**Key Components:**
```
- sticky-nav with nav-pills (horizontal)
- fast-hero or assessment-hero (large visual)
- timeline (horizontal with markers)
- stroke-comparison or condition-comparison (side-by-side cards)
- criteria-container with criteria-box (inclusion/exclusion)
- intervention-cards (2x2 grid)
- learning-objectives-grid (2-column cards)
```

**Existing Examples:** stroke.html

---

### 3. DASHBOARD/PANEL LAYOUT
**Used by:** Endocrine

**Characteristics:**
- Hormone level "dashboard" panels
- Feedback loop visualizations (hypothalamus → pituitary → target)
- Normal vs Abnormal value displays
- Hypo/Hyper comparison panels
- Medication timing charts
- Lab value gauges/meters

**Color Schemes:**
- Thyroid: Teal → Blue gradient
- Diabetes: Orange → Yellow gradient
- Adrenal: Purple → Pink gradient

**Key Components:**
```
- hormone-dashboard (panel grid)
- feedback-loop-diagram (visual flow)
- value-panel (normal range display)
- hypo-hyper-comparison (dual panels)
- medication-timing-chart
- symptom-checklist (checkboxes)
```

**Guides to Create:** diabetes-type1.html, diabetes-type2.html, thyroid-disorders.html, adrenal-disorders.html, pituitary-disorders.html

## Shared Elements Across All Layouts

All layouts share these common elements:

### Required Structure
```html
<!-- Download Bar (fixed at top) -->
<div class="download-bar">
    <div class="download-bar-brand">
        <i class="fas fa-book-medical"></i>
        <span>The Nursing Collective Study Guide</span>
    </div>
    <div class="download-bar-actions">
        <a href="../my-resources.html" class="btn-back">
            <i class="fas fa-arrow-left"></i> Back to Resources
        </a>
        <button class="btn-download" id="download-pdf-btn">
            <i class="fas fa-file-pdf"></i> Save as PDF
        </button>
    </div>
</div>

<!-- Document Container -->
<div class="document-container">
    <!-- Cover Page -->
    <div class="cover-page">...</div>

    <!-- Document Content -->
    <div class="document-content">
        <!-- Learning Objectives -->
        <!-- Navigation (style varies by layout) -->
        <!-- Content Sections -->
        <!-- Key Points Summary -->
    </div>

    <!-- Footer -->
    <div class="document-footer">...</div>
</div>

<!-- Script -->
<script src="guide-script.js"></script>
```

### Required Body Data Attributes
```html
<body data-product-id="guide-id" data-guide-name="Guide-Name">
```

### Required Info Box Types
All layouts should include these info box styles:
- `.info-box.warning` - Red/danger alerts
- `.info-box.clinical` - Yellow clinical pearls
- `.info-box.nclex` - Purple NCLEX tips
- `.info-box.tip` - Green helpful tips
- `.info-box.time` - Yellow/orange time-critical info

### Print Styles
All guides must include print-optimized CSS:
```css
@media print {
    .download-bar { display: none !important; }
    /* Page breaks, color preservation, etc. */
}
```

## Creating a New Guide

### Step 1: Identify the Category
Determine which category the guide belongs to from the list above.

### Step 2: Use the Correct Layout
Copy the structure from an existing guide in the same category, or follow the layout specification above.

### Step 3: Customize Colors
Each guide should have a unique gradient that fits the category theme but distinguishes it from other guides.

### Step 4: Add to Dashboard
Update `dashboard-script.js` to add the guide ID to the `htmlGuides` array:
```javascript
const htmlGuides = ['heart-failure', 'assessment-skills', 'myocardial-infarction', 'copd', 'stroke', 'NEW-GUIDE-ID'];
```

### Step 5: Test Print/PDF
Always test the print view (Cmd/Ctrl + P) to ensure proper page breaks and styling.
